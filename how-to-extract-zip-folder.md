## For MacOS
- Locate the ZIP file in Finder.
- Double-click on the ZIP file.
- macOS will automatically extract the contents into a folder with the same name as the ZIP file.

## For Windows
- Right-click the ZIP file.
- Select "Extract All"
- Choose a destination folder and click Extract.
